AI Code Error Analyzer

An AI-powered web application that analyzes code, detects errors, and provides intelligent suggestions — built completely from scratch as a full-stack project.

---

## 👩‍💻 About the Project

The **AI Code Error Analyzer** is designed to act like a coding mentor.
It helps developers identify mistakes in their code and understand how to fix them using AI-driven insights.

This project demonstrates my ability to build a **complete system independently**, including backend APIs, AI integration, and user-facing features.

---

## ✨ Key Features

* 🤖 AI-based code error detection and explanation
* 💡 Smart suggestions for fixing code issues
* 🧠 Mentor Chat for interactive AI assistance
* 📊 History tracking of analyzed code
* 📄 Report generation (TXT & PDF formats)
* 🔐 User authentication (Login/Signup/Session)
* ⚡ REST API architecture for scalability

---

## 🛠️ Tech Stack

**Frontend:**

* HTML, CSS, JavaScript

**Backend:**

* Django (Python)
* Django REST Framework

**AI Integration:**

* Ollama (LLM-based analysis system)

**Database:**

* SQLite

---

## 🧩 System Architecture

User → Frontend → Django Backend → AI Service → Response → User

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/AI-Code-Error-Analyzer.git
cd AI-Code-Error-Analyzer
```

### 2. Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run Server

```bash
python manage.py runserver
```
